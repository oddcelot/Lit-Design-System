StatCard is the big overview tile on the DevTools home dashboard — icon, large value, caption.

```html
<lit-stat-card icon="package" value="v3.2.1" label="Lit" accent></lit-stat-card>
<lit-stat-card icon="components" value="24" label="custom elements"></lit-stat-card>
```

Props: `icon`, `value`, `label`, `accent`.
