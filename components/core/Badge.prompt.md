Badge is a compact pill for statuses, counts, versions, and tags. Pick a `tone` from the semantic palette; add `dot` for a status indicator, `mono` for version/code values.

```html
<lit-badge tone="accent">reactive</lit-badge>
<lit-badge tone="success" dot>connected</lit-badge>
<lit-badge tone="neutral" mono>v3.2.1</lit-badge>
```

Props: `tone` (neutral · accent · cyan · success · warning · error), `dot`, `mono`.
