IconButton is a square, icon-only control — use in toolbars, panel headers, and the DevTools navigation rail. Always pass `label` for accessibility & tooltip.

```html
<lit-icon-button icon="settings" label="Settings"></lit-icon-button>
<lit-icon-button icon="components" active label="Elements"></lit-icon-button>
<lit-icon-button icon="refresh" variant="solid" label="Reload"></lit-icon-button>
```

Props: `icon`, `size` (sm · md · lg), `variant` (ghost · solid · accent), `active`, `label`.
