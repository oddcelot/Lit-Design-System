Button is the primary action control — use for any user-triggered action; flame-blue `primary` for the main action on a surface, `secondary`/`ghost` for lower emphasis, `danger` for destructive ones.

```html
<lit-button variant="primary" icon="plus">New element</lit-button>
<lit-button variant="secondary" size="sm">Filter</lit-button>
<lit-button variant="ghost" icon="refresh"></lit-button>
```

Props: `variant` (primary · secondary · ghost · danger), `size` (sm · md · lg), `icon` / `icon-right`, `full` (full-width), `disabled`.
